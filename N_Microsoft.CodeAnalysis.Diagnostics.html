AnalyzerFileReference
Represents analyzers stored in an analyzer assembly file. Analyzer are read from the file, owned by the reference, and doesn't change since the reference is accessed until the reference object is garbage collected. During this time the file is open and its content is read-only. If you need to manage the lifetime of the anayzer reference (and the file stream) explicitly use .
